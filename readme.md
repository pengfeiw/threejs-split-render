# Threejs 分屏绘制

使用 viewport 和 scissor 实现分屏渲染场景。

## Online Demo

[ThreeJs 分屏渲染场景在线查看](https://pengfeiw.github.io/threejs-split-render)

相关博客：[Threejs 分屏绘制](https://pengfeixc.com/blogs/computer-graphics/threejs-split-screen-render)

## How to run project

1. install package

```
npm install
```

2. start

```
npm run start
```

3. build

```
npm run build
```
